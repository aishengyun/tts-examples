## Configuration

Edit the `config.js`, set your `access_key` for our api. config `llm_url`, `llm_key` and `llm_model` for your own llm model.

## Run the app locally

```shell
npm i 
node file_output.js 
node file_output.js --sse 1
node llm_stream.js
```
